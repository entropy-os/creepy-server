# Creepy-Server
Creepy-Server is a server to host repositories for Creepy.

# Dependencies

To use Creepy-Server, you need :
  * Python3

# Using Creepy-Server

Open a terminal and type :

```git clone https://github.com/entropy-os/creepy.git && cd creepy-server && ./creepy-server```
